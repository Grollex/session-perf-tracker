# Gemini CLI Autonomous Agent

Этот проект позволяет запустить автономную копию Gemini CLI в вашей системе.

## Инструкция по запуску:

1. **Node.js**: Убедитесь, что у вас установлен Node.js (https://nodejs.org/). Если его нет, программа предложит открыть страницу загрузки.
2. **API Ключ**: Вам понадобится ключ API от Google AI Studio. 
   - Получить его можно бесплатно здесь: [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
3. **Запуск**: 
   - Запустите `GeminiAgent.exe`.
   - При первом запуске вставьте ваш API ключ. Он будет сохранен в `config.json`.
   - Программа автоматически установит необходимые пакеты и запустит агента.

## Структура:
- `GeminiAgent.exe`: Загрузчик.
- `GeminiAgent.ps1`: Логика установки и запуска.
- `config.json`: (Создается при запуске) Хранит ваш API ключ.
