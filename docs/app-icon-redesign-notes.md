# App Icon Redesign Notes

Next icon pass should target taskbar readability first.

- Use transparent background.
- Do not include a square background tile inside the icon.
- Let the symbol itself fill most of the available square.
- Keep the silhouette simple enough to read at 16-32 px.
- Communicate monitoring / analytics / session performance.
- Avoid visual language that reads like a system booster, cleaner, or optimizer.
- Prefer one clear mark over detailed dashboards, tiny graphs, or dense UI panels.
